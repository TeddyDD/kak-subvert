# kak-subvert

Allow converting selection to various casing.

For instance, `foo_bar` to `FooBar`

Known casings:

```
camel               fooBar
kebab               foo-bar
pascal              FooBar
sentence            Foo bar
snake               foo_bar
screaming           Foo bar
title               Foo Bar
train               Foo-Bar
ugly                Foo_Bar
```
